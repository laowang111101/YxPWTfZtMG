# 基于Python的图书馆管理系统 python313

## 项目概述

本项目是一个基于Python语言的图书馆管理系统，使用Django框架开发，以Mysql数据库作为后端存储。提供完整的源代码、数据库结构、报告文档以及全过程开发讲解，确保开发运行环境齐全。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: Mysql

## 功能模块

系统主要包含以下功能模块：

1. 用户管理
   - 用户注册与登录
   - 用户权限管理
2. 图书管理
   - 图书信息录入与修改
   - 图书借阅与归还
3. 系统查询
   - 图书检索
   - 借阅记录查询
4. 统计分析
   - 图书借阅排行榜
   - 用户借阅统计

## 系统角色

- 系统管理员：负责系统维护、用户权限管理等
- 普通用户：进行图书检索、借阅与归还等操作

## 运行环境

- Python环境
- Django框架
- Mysql数据库

## 部署步骤

1. 配置Python环境
2. 安装Django框架
3. 配置Mysql数据库
